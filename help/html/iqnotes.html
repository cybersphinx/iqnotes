Introduction

IQNotes is notes kept in a hierarchical manner. This is a simple and handy way to keep data well organized. Each note can be tasked with percentage complete, priority or event with day, month, year and optionally time. IQNotes also provides advanced sorting. Each note can additionally be associated with entries like company, person, user account, credit card, WWW link ... and even a sketch.
Project page: http://iqnotes.berlios.de
Author: Peter Vrabel, kybu@kybu.org

Menu

File

New
Create a new file.
Open
Open a data file.
Save
Save a data file.
Close
Close the current data file.
Quit
Quit the application

Tree

Search
Search the tree for a string. The results will be shown in a new tree, which can then be closed. After closing, the current note in the search tree is set as the current note in the tree.
Quick add
Will add a text note at the top of the tree, with the name 'Quick note'. Useful for fast adding.
Add before
Add a new note before the selected note.
Add after
Add a new note after the selected note.
Add child
Add a new note as a child of the selected note.
Sort
You can set the order priorities for each note type (task, event, ...), define the show order (if the tasks are first event, second ...). You can sort in the current level, in the current level and down, or the whole tree. The settings in the sort dialog are remembered for the next use.
Expand tree
Expand the whole tree.
Collapse tree
Collapse the whole tree.
Task list
Show tasks in a flat view. Tasks can be taken from the whole tree or from the current note down. Tasks in this list can be sorted. When closing this list, the current task in this list is selected as current in the notes tree.
Event list
Same as Task list, but for events
Reminder
Show the reminder list

Note

Edit
Edit the selected note.
Cut
Delete the selected note and store it in the clipboard. This function is useful for removing or moving a note and its subtree to another place.
Copy
Copy only the current note or also it's whole subtree to the clipboard.
Paste
Paste the note stored in the clipboard after/before/as a child of the current note.
This is a potentially dangerous action if you are moving a note from another data file, if that file has a different definition for its entries, with different properties than are in the current file. So before moving from another data file, check the definition in the Options menu to see if they are the same in both data files. If you had to make changes to the entry to make it the same, make the changes, save and close that data file, open it again, delete the note you wish to move, open destination data file and add the last deleted note. Pretty confusing, isn't it? ':)
Set picture
Set the picture for the current note. See IQNotes homepage for additional nice pictures. If you want to add your own pictures, just load them to /home/QtPalmtop/iqnotes/items. The pic size _MUST_ be 16x16.
Set task
Set the task properties (percentage, priority) for the current note.
Set event
Set the event properties (date and optionally time) for the current note.
Unset
Discard the task or event data of the current note.
Set reminder
Set the reminder for the current note. Set the date when the item is shown in the reminder list.
Unset reminder
Discard the reminder associated with the current note.

Options

Define new entry
The note can be plain text, or a defined entry. You can define any kind of entry, for example 'Pet'. You must define the properties, like 'Name', 'Color', 'Count of the legs', 'Personality', ... :). Notes added thereafter will prompt for the new data item.
Change entry
If you want to change an entry, click here. Even the entry which is used in notes can be changed.
Delete entry
Delete an entry. Only entries which are not used by IQNotes can be deleted.

Notes

Select if the root node should have a plus sign (+), if it has children.
Word wrap
If word wrap is set, than text the widget is set to word wrap notes.
Show scrollbars
Show or hide a scrollbars.
Vertical layout
Use vertical or horizontal layout. Set the most suitable for you.
Choose default font
Backup
This option, if set, automatically saves a backup file in a chosen location after IQNotes is closed. The location must be a directory, for example '/mnt/cf/backup/iqnotes'. Currently, there is only a manual method to re-use a backup file. You have to copy the backup file to the location ~/Applications/iqnotes/
Security
Here, you can set up encryption for the saved data file. The data is encrypted by rijndael aka AES, which the USA government considers as secure and it replaced the DES algorithm. If you haven't set up a password the first two options (current, don't encrypt) will be disabled. With the next two options, you can set up or change your password. For changing or turning encryption off, you must supply the current password. The backup file is also stored encrypted. BEWARE, there is no way to decrypt data if you forgot the password.

Tip and tricks

Sketches
If is the sketch too big, you can move it in the half or hide tree view simply by tapping on it and moving the stylo to the desired direction.
View layout
Are you using Zaurus C7xx model? Try to change layout in the Options / Preferences for better viewing.
